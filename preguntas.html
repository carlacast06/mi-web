PREGUNTAS

Qué es una red informática.

Es un conjunto de equipos conectados por medio de cables, señales, ondas o cualquier otro método de transporte de datos, que comparten información (archivos), recursos (CD-ROM, impresoras)

Qué es una red LAN.

Red de comunicación entre ordenadores situados en el mismo edificio o en edificios cercanos, de forma que permite a sus usuarios el intercambio de datos y la compartición de recursos

Qué es una red WAN.

Es la tecnología que conecta entre sí a las oficinas, los centros de datos, las aplicaciones en la nube y el almacenamiento en la nube

Qué es Internet

Red global de redes de ordenadores cuya finalidad es permitir el intercambio libre de información entre todos sus usuarios

Qué es la Web

Es un sistema interconectado de páginas web públicas accesibles a través de Internet

Qué es WWW

Es un sistema interconectado de páginas web públicas accesibles a través de Internet

Qué es el protocolo HTTP y HTTPS

Los mensajes HTTP son de texto plano, lo que significa que las personas no autorizadas pueden acceder a ellos y leerlos fácilmente a través de Internet. Por el contrario, HTTPS transmite todos los datos de forma cifrada

Página web estática vs página web dinámica

A diferencia de las páginas web estáticas, que muestran el mismo contenido para todos los usuarios, las páginas web dinámicas pueden personalizarse y cambiar en tiempo real

CMS. Ejemplos. Ventajas e inconvenientes

Es un sistema online que permite poner en marcha una página web de forma sencilla y rápida Ejemplo: un blog, un ecommerce o cualquier tipo de página web Ventajas: Con un CMS puedes ahorrarle tiempo al equipo de desarrollo, y actualizar directamente el sitio. Desventajas:Es que consumen grandes recursos del servidor,dado que en cada llamada, el servidor tiene que cargar el motor, los componentes, los plugins y la base de datos del sistema de gestión de contenidos, lo cual repercute negativamente en la velocidad del sitio web.

Proveedor de servicios de Internet. Ejemplos

Algunos ejemplos de ISP en distintos países son: España. Movistar, Vodafone, Orange, Yoigo, R (telecomunicaciones), Euskaltel, Telecable, Adamo Telecom

Dirección IP. Diferencia entre una dirección IP privada y pública.

Una dirección IP privada se utiliza dentro de una red privada para conectarse de forma segura a otros dispositivos dentro de esa misma red. Una dirección IP pública te identifica ante el resto de Internet para que toda la información que buscas pueda encontrarte

¿Qué es el alojamiento web/hosting? Ejemplos

Un hosting es un servicio de alojamiento web que te permite publicar un sitio web o aplicación en Internet
